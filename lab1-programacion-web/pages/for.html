Bucle for en PHP

Ideal cuando conoces la cantidad de iteraciones. Tiene inicio, condición y actualización.

Sintaxis básica

<?php
$suma = 0;
for ($i = 1; $i <= 10; $i++) {
  $suma += $i;
}
echo $suma; // 55

Ejemplo breve

Generar una lista HTML de 1 a 5.

<?php
echo "<ul>";
for ($i = 1; $i <= 5; $i++) {
  echo "<li>Elemento $i</li>";
}
echo "</ul>";
Salida simulada: <ul><li>Elemento 1</li>...<li>Elemento 5</li></ul>

Errores comunes

  • Condición incorrecta (se queda corto o se pasa).
  • Olvidar actualizar el contador.
