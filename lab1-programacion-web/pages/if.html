Estructura if en PHP

El if evalúa una condición. Si se cumple, ejecuta un bloque; si no, lo ignora o pasa a else/elseif.

Sintaxis básica

<?php
$edad = 19;

if ($edad >= 18) {
  echo "Acceso permitido";
} else {
  echo "Acceso denegado";
}

Ejemplo breve

Aplicar 20% de descuento si la persona es estudiante.

<?php
$precio = 10.00; 
$esEstudiante = true;

if ($esEstudiante) {
  $precio = $precio * 0.8; // 20% de descuento
}
echo "Total a pagar: $" . number_format($precio, 2);
Salida simulada: Total a pagar: $8.00

Errores comunes

  • Usar = en lugar de == o ===.
  • Olvidar llaves cuando el bloque crece.
  • Comparar números/cadenas sin validar tipos (mejor === cuando aplica).
