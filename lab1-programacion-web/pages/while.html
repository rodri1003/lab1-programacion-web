Bucle while en PHP

Repite un bloque mientras la condición sea verdadera. Útil cuando no sabemos cuántas iteraciones habrá.

Sintaxis básica

<?php
$i = 1;
while ($i <= 5) {
  echo $i . " ";
  $i++;
}
// Salida: 1 2 3 4 5

Ejemplo breve

Construir una cadena con números del 1 al 4.

<?php
$i = 1;
$texto = "";

while ($i <= 4) {
  $texto .= $i;
  if ($i < 4) { $texto .= ", "; }
  $i++;
}
echo $texto;
Salida simulada: 1, 2, 3, 4

Errores comunes

  • Olvidar actualizar el contador → bucle infinito.
  • Condición mal planteada (no se cumple nunca o siempre).

Mini‑ejercicio
