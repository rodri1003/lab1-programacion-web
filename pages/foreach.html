Bucle foreach en PHP

Sirve para recorrer arreglos y colecciones de manera simple y legible, sin manejar índices manualmente.

Sintaxis básica

<?php
$numeros = [10, 20, 30];
foreach ($numeros as $n) {
  echo $n . " ";
}
// Salida: 10 20 30

Ejemplo breve

Recorrer productos y mostrar nombre y precio.

<?php
$productos = [
  ["nombre" => "Cuaderno", "precio" => 2.5],
  ["nombre" => "Lápiz",    "precio" => 0.5],
  ["nombre" => "Mochila",  "precio" => 15.0],
];

foreach ($productos as $p) {
  echo $p["nombre"] . ": $" . number_format($p["precio"], 2) . "\\n";
}
Salida simulada: Cuaderno: $2.50
Lápiz: $0.50
Mochila: $15.00

Errores comunes

  • Confundir clave/valor cuando el arreglo es asociativo.
  • Modificar el arreglo dentro del foreach sin cuidado.
