Estructura switch en PHP

Permite elegir entre varios caminos según el valor de una variable. Útil cuando hay varios casos posibles.

Sintaxis básica

<?php
$dia = "lunes";

switch ($dia) {
  case "lunes":
    echo "Ánimo, empieza la semana";
    break;
  case "viernes":
    echo "Casi fin de semana";
    break;
  default:
    echo "Día normal";
}

Ejemplo breve

Mensaje según rol de usuario.

<?php
$rol = "editor";

switch ($rol) {
  case "admin":
    echo "Tienes acceso total";
    break;
  case "editor":
    echo "Puedes crear y editar contenido";
    break;
  case "visitante":
    echo "Solo lectura";
    break;
  default:
    echo "Rol no reconocido";
}
Salida simulada: Puedes crear y editar contenido

Errores comunes

  • Olvidar break (cae al siguiente caso).
  • No incluir default para manejar valores no contemplados.

Mini‑ejercicio

Según el mes (“enero”…“diciembre”), muestra si pertenece a “Primer trimestre”, “Segundo…”, etc.
